### Обновленная документация языка Zmey Gorynich

Текущая версия языка на основе примера от 4 апреля 2025 года. Документация обновлена с учетом добавления поддержки фиксированных циклов (`дважды`, `трижды` и т.д.), а также исправлений и дополнений, основанных на текущем состоянии кода.

---

## Запуск программы
Программа запускается командой:
```plaintext
python -m logic/interpreter.py examples/program.zg
```

## Основные особенности
- Каждая команда вне блока кода заканчивается словом `гойда`.
- Блоки кода выделяются конструкциями `ухожу я в пляс` и `закончили пляски`.
- Поддерживаются базовые типы данных, операции, списки, условные операторы, циклы и функции.
- Добавлена поддержка фиксированных циклов (`дважды`, `трижды` и т.д.).

## Базовые команды и функции

### Вывод данных
- `молвить(значение)` — выводит значение в консоль.  
  Может принимать несколько аргументов через запятую или одиночные значения (строки, числа, переменные, выражения).

Примеры:
```zmeygorynich
молвить("Славянский зажим яйцами") гойда  # Выводит строку
молвить(5 + 9) гойда                     # Выводит 14
молвить("Число: ", число) гойда          # Выводит "Число: " и значение переменной число
молвить(ящер) гойда                      # Выводит значение переменной ящер
```

### Ввод данных
- `внемли переменная` — считывает ввод пользователя и присваивает его переменной.

Пример:
```zmeygorynich
внемли ящер гойда
молвить ящер гойда  # Выводит введенное значение
```

### Объявление переменных
Переменные могут объявляться с указанием типа через `быти` или без явного указания типа (динамическая типизация). Поддерживаемые типы:
- `быти цело` — целое число.
- `быти плывун` — число с плавающей точкой (двойная точность по умолчанию).
- `быти строченька` — строка.
- `быти список` — список (может быть типизирован, например, `список цело`).
- Специальные типы чисел с повышенной точностью:
  - `быти плывун малый точный` — число с точностью 30 знаков.
  - `быти плывун великий` — число с точностью 50 знаков.
  - `быти плывун звездный` — число с точностью 100 знаков.

Примеры:
```zmeygorynich
ящер быти цело = 5 гойда
ящер быти плывун = 5.5 гойда
ящер быти строченька = "5" гойда
массив быти список строченька = ["первая", "вторая"] гойда
мера быти плывун малый точный = 1.1234567890123456789012345678901234567890 гойда
круг быти плывун великий = 3.141592653589793238462643383279502884197169399375105820974944 гойда
звѣзды быти плывун звездный = 1.6180339887498948482045868343656381177203091798057628621354486227052604628189 гойда
```

Без явного указания типа:
```zmeygorynich
ящер = 100 + 9 гойда  # ящер будет целым числом (109)
```

## Арифметические операции
Поддерживаются следующие операции:
- `+` — сложение.
- `-` — вычитание.
- `*` — умножение.
- `/` — деление.
- `%` — остаток от деления.
- `**` — возведение в степень.
- `корешок из` — квадратный корень.

Примеры:
```zmeygorynich
ящер = 100 + 9 гойда    # 109
ящер = 100 - 9 гойда    # 91
ящер = 100 * 9 гойда    # 900
ящер = 100 / 9 гойда    # 11.111...
ящер = 100 % 9 гойда    # 1
ящер = 100 ** 2 гойда   # 10000
ящер = (10 ** 10) / (корешок из 100) гойда  # 316227766.016...
```

## Строки
- Конкатенация строк выполняется через `+`.

Пример:
```zmeygorynich
ящер = "ящер" + "гойда" гойда  # "ящергойда"
```

## Списки
- Объявление списка: `имя = [элемент1, элемент2, ...] гойда`.
- Создание списка с заданным размером и значением: `созвать_дружину(длина, значение) гойда`.
- Доступ по индексу: `имя[индекс]`.
- Присваивание по индексу: `имя[индекс] = значение гойда`.
- Конкатенация списков: `+`.

Примеры:
```zmeygorynich
МатушкаЗемля = [1, 2, 3] гойда
массив = созвать_дружину(3, 0) гойда    # [0, 0, 0]
массив[1] = 42 гойда                    # [0, 42, 0]
молвить(массив[1]) гойда                # Выведет: 42
массив2 = [1, 2, 3] гойда               # [1, 2, 3]
массив3 = массив + массив2 гойда        # [0, 42, 0, 1, 2, 3]
молвить(массив3) гойда                  # Выведет: [0, 42, 0, 1, 2, 3]
```

## Условные операторы
- `аще условие то` — if.
- `аще ли условие то` — elif.
- `ино` — else.

Операторы сравнения:
- `==` — равно.
- `>` — больше.
- `<` — меньше.

Пример:
```zmeygorynich
ящер быти цело = 10 гойда

аще ящер == 10 то ухожу я в пляс
    молвить("Ящер ровно 10!") гойда
закончили пляски

аще ли ящер > 5 то ухожу я в пляс
    молвить("Ящер велик!") гойда
закончили пляски

ино ухожу я в пляс
    молвить("Ящер мал!") гойда
закончили пляски
```

## Циклы

### Цикл `покуда` (while)
- `покуда условие` — выполняет блок кода, пока условие истинно.

Пример:
```zmeygorynich
ящер = 0 гойда
рептилия = 1 гойда

покуда ящер < 10 ухожу я в пляс
    ящер = ящер + 1 гойда
    рептилия = рептилия * ящер гойда
    молвить("ящер: ", ящер) гойда
    молвить("рептилия: ", рептилия) гойда
закончили пляски
```

### Фиксированные циклы
Добавлена поддержка фиксированных циклов, которые выполняют блок кода или одиночный оператор заданное количество раз. Поддерживаемые ключевые слова:
- `дважды` — 2 раза.
- `трижды` — 3 раза.
- `четырежды` — 4 раза.
- `пятьжды` — 5 раз.
- `шестьжды` — 6 раз.
- `семьжды` — 7 раз.
- `осьмьжды` — 8 раз.
- `девятьжды` — 9 раз.
- `десятьжды` — 10 раз.
- `стожды` — 100 раз.

Фиксированный цикл может содержать:
- Блок кода (с `ухожу я в пляс` и `закончили пляски`).
- Одиночный оператор, за которым следует `гойда`.

Примеры:
```zmeygorynich
# Цикл с блоком кода
ᚢ быти цело = 0 гойда
десятьжды ухожу я в пляс
    ᚢ = ᚢ + 1 гойда
    молвить(ᚢ) гойда
закончили пляски  # Выведет числа от 1 до 10

# Цикл с одиночным оператором
пятьжды молвить("ᚢ") гойда  # Выведет "ᚢ" 5 раз
```

## Функции
- Объявление: `сотвори имя(параметры) изречет тип`.
- Возврат значения: `возверни значение`.

Примеры:
```zmeygorynich
сотвори факториал(n) изречет цело ухожу я в пляс
    результат = 1 гойда
    покуда n > 1 ухожу я в пляс
        результат = результат * n гойда
        n = n - 1 гойда
    закончили пляски
    возверни результат
закончили пляски

сотвори возвысь(основа, степень) изречет плывун ухожу я в пляс
    возверни основа ** степень
закончили пляски

результат = факториал(5) гойда  # 120
молвить(результат) гойда

результат = возвысь(2, 3) гойда  # 8.0
молвить(результат) гойда
```

## Поддержка Unicode
Язык поддерживает Unicode-символы в именах переменных и строках.

Примеры:
```zmeygorynich
молвить("ᚠᚢᚦᚨᛚᛏ") гойда  # Выведет строку с рунами
ᚠ быти плывун = 2.3 гойда
молвить(ᚠ) гойда  # Выведет 2.3
```
